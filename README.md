Cattle Business Website

Simple, responsive website for a cattle farm built with pure HTML and JavaScript.

Features

· Mobile-friendly design
· Interactive cattle gallery
· Contact form with validation
· Farm information display
· No frameworks or dependencies

Setup

1. Download index.html
2. Create an assets folder for images
3. Add your cattle photos
4. Open in any browser

Customization

Edit the HTML file directly:

· Update business name and contact info
· Replace placeholder images
· Modify farm details
· Adjust colors in the CSS section

That's it! No build process or installation needed.
